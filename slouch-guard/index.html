Deep Technical Architecture

Under The Hood Engineering

Built ground-up on WinUI 3 and .NET 8 for high performance and total privacy.

1. Native Computer Vision Engine

Unlike heavy multi-gigabyte Python or deep-learning wrappers that consume massive RAM and drain laptop battery, SlouchGuard utilizes Windows native hardware acceleration:

  • Windows Native Face Analysis: Leverages Windows.Media.FaceAnalysis.FaceDetector and MediaFrameReader for rapid, low-latency frame sampling.
  • Optimized Pixel Formats: Processes frames in BitmapPixelFormat.Gray8, ensuring minimal CPU/GPU impact even while running high-FPS games or CAD software.
  • 100% On-Device Volatile Processing: Video frames are analyzed transiently in volatile RAM and immediately discarded. Zero disk logging and zero cloud transmission.

2. Triple-Vector Posture Algorithms

SlouchGuard combines dynamic baseline calibration with a multi-dimensional tracking algorithm to prevent false positives:

  • Vertical Slouching: Computes downward displacement of head coordinates relative to the calibrated optimal baseline height.
  • Lateral Tilt & Lean: Identifies side-to-side drooping, uneven shoulder weight distribution, and off-center spinal curvature.
  • Depth & Proximity ("Tech Neck"): Measures facial bounding-box scale expansion relative to camera distance to catch unhealthy forward neck leaning.
  • EMA & Median Filtering: Applies Exponential Moving Average (EMA) and temporal median filtering to eliminate camera sensor noise and natural head nods.

Win32 Glow Window Overlay

Built with native Win32 window styles (WS_EX_LAYERED, WS_EX_TRANSPARENT, WS_EX_TOPMOST, WS_EX_NOACTIVATE). Projects subtle perimeter glows in Red, Orange, Blue, or Purple that stay transparent and never steal keyboard or mouse focus during full-screen work.

Stretch & Break Automation

Features automated 20-20-20 eye rest and periodic stretch reminders (e.g., every 30 mins). Supports customizable sound alerts, system toast popups, and quick snooze buttons (15m, 30m, 1h).

Source-Generated C# JSON

Uses compiled C# JSON source generators (JsonSerializerContext) for zero-reflection, instant loading of local settings and historical log data without memory allocation spikes.
