Slouch Guard

🪑 Sit Better. Feel Better. Work Better. Real-Time AI Posture Corrector

SlouchGuard is a lightweight, privacy-first desktop application that monitors your sitting posture in real-time using your webcam. It gently alerts you when you slouch or lean forward—preventing back and neck strain.

AI Core Engineering

Next-Gen Desk Health Features

Real-Time Detection

Tracks slouch vertical drop and forward head proximity using Windows Native FaceDetector API with GPU acceleration.

100% Local Privacy

No cloud server, no accounts, and no internet connection required. Camera frames are processed in memory and discarded instantly.

Customizable Notifications

Choose toast popups, subtle alarms, or screen edge glow (5 vibrant color themes: Red, Amber, Neon, Blue, Rainbow).

Posture Analytics & History

Track your progress with weekly ratio charts, hourly posture trends, and habit summary metrics to build lasting posture discipline.

Fine-Tuned Control

Adjust sampling intervals from 3 to 30s, set 5 sensitivity levels, customize alert delays, or pause alerts during meetings.
